ABOUT US


Welcome to our site!
We are a dedicated team passionate about technology and data analysis. Our project, "Development of a Web Application for Analyzing Airline Customer Reviews Using Web Scraping and Data Visualization Methods," aims to provide deep insights and visualizations of airline customer reviews to help companies improve their service quality and customer satisfaction.

Our Mission
Our goal is to create a tool that enables airlines and their customers to better understand each other by offering valuable insights and clear data visualizations for informed decision-making and enhanced service experiences.

Technologies We Use
- Programming Language: Python - for web scraping, data analysis, and processing.
- Framework: Flask - for building a flexible and reliable web application.
- Development Environment: Visual Studio Code - for writing, testing, and debugging code.
- Data Analysis Libraries: Pandas, Numpy, NLTK - for processing and analyzing reviews.
- Data Visualization Libraries: Matplotlib, Seaborn - for creating clear and insightful graphs and charts.


What We Offer
Our web application allows you to:
- Collect customer reviews from various online sources.
- Analyze text data to identify key themes and sentiments.
- Visualize analysis results in an accessible and understandable format.
We believe our project will help airlines improve their services and customers enjoy better experiences. Thank you for visiting our site, and we hope our tool will be useful to you.
